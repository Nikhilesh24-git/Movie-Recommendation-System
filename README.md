# Movie Recommendation System 🎬

## 🚀 Features

- 🎯 User ratings input
- 🤝 Collaborative filtering (user–user or item–item)
- 🧠 Content-based recommendations (genre, keywords)
- 📊 Display top-N recommendations
- 🔄 Train/test split & performance metrics

- **Language:** Python
